Data Processing Unit Creation

To create a data processing unit (DPU), you should:
  1. Study basic concepts and technologies
  2. Follow the DPU creation tutorial

List of available Data Processing Units

Core data processing units (DPUs) available with every distribution of ODCleanStore:

  • RDF File Extractor - extracts data from RDF file
  • SPARQL Endpoint Extractor - extracts data from certain SPARQL endpoint
  • SPARQL Transformer - executes defined SPARQL queries
  • RDF File Loader - loads resuling data to RDF file
  • SPARQL Endpoint Extractor - loads resulting data to target SPARQL endpoint
  • Silk Linker - creates links between resources, using Silk (under preparation)
  • Data Fusion - creates links between resources, using the ODCS-Fusion tool (under preparation)
